# Installation

 Step-by-step instructions to seamlessly set up the LLMP framework on your system.
 Whether you're a seasoned developer or just starting out, this guide ensures a smooth installation process,
 letting you dive into LLMP's capabilities without a hitch.

---

### **Prerequisites**

Ensure you have:

1. Python 3.8 or higher.
2. An environment suitable for Python projects.

---

### **Installation Process**

Install LLMP with pip:

```bash
pip install pyllmp
```

or with git:

```bash
git clone
```

---



### **Verification**

After installation, verify the setup:

```python
import llmp
print(llmp.__version__)
```

This will display the LLMP version you've installed.
